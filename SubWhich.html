Which เป็นประธาน (Subject Which Questions)

โครงสร้างประโยค

Which + Verb (กริยาช่องที่ 1) + Object/Rest of sentence?

หมายเหตุ: ไม่ต้องใช้ Helping Verb (do, does, did) เพราะ Which ทำหน้าที่เป็นประธาน

ลักษณะเฉพาะของ Which

  • ใช้ถามเมื่อมีตัวเลือกจำกัด
  • ผู้ถามและผู้ตอบต้องรู้ตัวเลือกที่เป็นไปได้
  • มักใช้กับสิ่งของมากกว่าบุคคล
  • สามารถตามด้วย of + กลุ่มตัวเลือก

ตัวอย่างการใช้ Which เป็นประธาน

Which of these cars belongs to you?

รถคันไหน (จากรถเหล่านี้) เป็นของคุณ

Which works better for this problem?

วิธีไหนใช้ได้ดีกว่ากับปัญหานี้

Which of the answers is correct?

คำตอบข้อไหน (จากคำตอบเหล่านี้) ถูกต้อง

การเปรียบเทียบ Which vs What

Which What
ใช้เมื่อมีตัวเลือกจำกัด ใช้เมื่อไม่จำกัดตัวเลือก
Which book is yours? (จากหนังสือที่เห็น) What book do you want? (จากหนังสือทั้งหมดที่มีอยู่)
Which of these colors do you prefer? What color do you like?

รูปแบบการใช้ Which ที่พบบ่อย

1. Which of + กลุ่มคำนามพหูพจน์

Which of these students passed the exam?

นักเรียนคนไหน (จากนักเรียนเหล่านี้) สอบผ่าน

2. Which one + (of + กลุ่มคำนาม)

Which one of the options suits you best?

ตัวเลือกไหน (จากตัวเลือกเหล่านี้) เหมาะกับคุณที่สุด

ข้อควรจำ

  • Which ใช้เมื่อมีตัวเลือกที่ชัดเจน
  • สามารถใช้ "of" ตาม Which เพื่อระบุกลุ่มตัวเลือก
  • กริยาที่ตามหลังต้องสอดคล้องกับประธานที่แท้จริง
  • Which สามารถใช้กับทั้งสิ่งของและบุคคล แต่นิยมใช้กับสิ่งของมากกว่า

แบบฝึกหัด

สร้างประโยคคำถามโดยใช้ Which เป็นประธาน:

1. (มีรถ 3 คัน) The red car belongs to me.

→ Which of these cars belongs to you?

2. (มีหนังสือ 2 เล่ม) The blue book costs more.

→ Which book costs more?

3. (มีทางเลือก 2 ทาง) This solution works better.

→ Which solution works better?
