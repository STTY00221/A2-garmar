การใช้ So กับ Auxiliary Verbs

โครงสร้าง

So + Auxiliary Verb + Subject

ใช้เพื่อแสดงว่าเราทำหรือรู้สึกเหมือนกับที่คนอื่นพูดมา

การใช้ So กับ Auxiliary Verbs ต่างๆ

I am tired!
So am I.
I like studying.
So do I.
John loves music.
So does Mary.
I was at the office.
So was I.
Auxiliary Verb การใช้งาน ตัวอย่าง
am/is/are สำหรับ Present BE I am happy. → So am I.
was/were สำหรับ Past BE You were late. → So was I.
do/does สำหรับ Present Simple I like coffee. → So do I.
did สำหรับ Past Simple I went home. → So did I.
have/has สำหรับ Present Perfect I have eaten. → So have I.
will/would สำหรับ Future/Conditional I will go. → So will I.
can/could สำหรับ Ability/Permission I can swim. → So can I.
should/must สำหรับ Obligation/Necessity You should study. → So should I.

หมายเหตุสำคัญ

  • ต้องใช้ Auxiliary Verb ตัวเดียวกับประโยคแรก
  • Subject ต้องอยู่หลัง Auxiliary Verb เสมอ
  • สามารถใช้ too แทน So ได้ แต่จะอยู่ท้ายประโยค
  • ระวังการใช้ Subject ให้ถูกต้อง (I, you, he, she, they)
